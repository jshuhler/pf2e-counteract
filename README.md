# pf2e-counteract
A tool for calculating the outcome of Pathfinder 2e counteract checks using the [Pathfinder Second Edition Remastered rules](https://2e.aonprd.com/Rules.aspx?ID=3280).


## Why?
I am terrible at these rules in game. I am learning Javascript. This taught me the rules and let me work on a personal project for fun on the side. 

This extra practice helped brush up on passing variables and values between multiple functions and let me continue practicing with IIFEs and objects.
