# pf2e-counteract
A [tool](https://jshuhler.github.io/pf2e-counteract/) for calculating the outcome of [Pathfinder 2e](https://paizo.com/) counteract checks using the [Pathfinder Second Edition Remastered rules](https://2e.aonprd.com/Rules.aspx?ID=3280).

## How?
A little bit of Javascript and some HTML and CSS. Let me know if you find an error in a calculation.

## Why?
I am terrible at these rules in game and I am learning Javascript. 

This taught me the rules, let me work on a personal project for fun on the side, gave me some much needed repetition on passing variables and values between multiple functions, and let me continue practicing with IIFEs and objects.
